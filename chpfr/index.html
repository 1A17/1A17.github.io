CHP First Response - LSPDFR Brought To FiveM

The purspose of CHP First Response is to develop a great LSPDFR like script that is a standalone single script. This amazing script by CHPOfficer20 has an iconic AI Backup including Code Three, Code Two, Traffic Assist, and Pursuit Backup. These assists can come in any form or vehicle. The script also includes a brand new callout system that is unlike any other! If you would like to give feedback or/and help in the development in this script, feel free to join the discord! There are so many opportunities for this script and we are attempting to develop it to its complete max. If we can reach our wanted status, the script will act as an already modded LSPDFR but with multiplayer due to its presence on FiveM. The script is not yet completed because it has much more work to go and will be an amazing creation when it reaches our expectations at CHPOfficer20!

CHP-FR Features:

  • AI Interactions (Arrest, Pullover, Other)
  • Vehicle Interaction (Tow, Search, Other)
  • In-Game CAD/MDT System (Synced with AI Names)
  • Callout System (Latest and Random 911 Calls from AI)
  • AI Backup (NPC Police Backup)
    • Code 3 Assistance (State, County, City)
    • Code 2 Assistance (State, County, City)
    • Traffic Stop Assistance (State, County, City)
    • Code 5 Assistance (State, County, City)
    • AI Pursuit Assistance (State, County, City)
  • AI Police Partner Script (A police partner will follow you, assist you, and you have a list of actions that you can have your partner do.
